# Spoke (alpha)

Easily create custom 3D environments for [Mozilla Hubs](https://hubs.mozilla.com).

## Usage

- [Download](https://github.com/MozillaReality/Spoke/archive/master.zip) and extract this repo
- `cd Spoke-master`
- `npm ci`
- `npm run dev`

Then open http://localhost:9090.
