# Spoke (beta)

Easily create custom 3D environments for [Mozilla Hubs](https://hubs.mozilla.com).

## Development

- `git clone https://github.com/MozillaReality/Spoke.git`
- `cd Spoke`
- `npm ci`
- `npm run dev`

Then open http://localhost:9090.
